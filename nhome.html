About GHOST-SH-bar

GHOST-SH-bar is a Web Penetration tool built for Android Devices where you can perform SQL Injection, Local File Inclusion (LFI), Cross Site Scripting (XSS), Shell Uploading, Defacement & many more.

It is a modified version of Anon HackBar, originally developed by MasterJeo and later modified by SANATANI_x_ANONYMOUS & Ghost Op of SANATANI HACKERS Community.
