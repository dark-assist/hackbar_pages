GHOST-SH-bar

Based & Inspired by
Mozilla HackBar Addon

Developed By
SANATANI_x_ANONYMOUS & Ghost Op

Official Instagram

GHOST-SH-bar is a web penetration tool built for android devices where one can use this tool for SQL Injection (SQLI), File Inclusion (LFI), Cross Site Scripting (XSS), Shell Uploading, Defacement and many more. Inspired by Mozilla HackBar Addon but with such more advance features.
